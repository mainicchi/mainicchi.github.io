The long story

This projects main task was to work on the Likeit system, a comprehensive tool utilized by corporate clients for demanding workforce and task management. Key functionalities include client and employee data management, task coordination, recruitment, time tracking, collective agreement interpretation, payroll, invoicing, and accounting.

The project narrowed its design scope to the employee view, addressing mobile visibility and responsiveness as primary concerns. The emphasis was on redesigning the system with a mobile-first approach, particularly focusing on time tracking functionalities.

The old system and how it looks to the user.

Process

Leveraging service design methods, including user testing, the project generated proposals for future time-tracking interfaces. Valuable insights from the client aided in refining designs, resulting in Figma-based proposals and a comprehensive report for future development.

The research phase involved system exploration, benchmarking, card sorting, and on-site workshops with the client. Also Figma facilitated wireframing, followed by light user testing on the wireframes which made this project a very good learning experience for me on facilitating. The visual prototypes were created in Figma, tested, and refined based on user test feedback.

Teams meetings were held weekly throughout the project, ensuring continuous communication and collaboration with the client. This was crucial because the system we were dealing with was both unfamiliar and outdated, underscoring the necessity to grasp its functionality in order to conduct effective testing.

The design process included benchmarking, user mapping, a workshop and card sorting for user testing

As a UX designer, I created wireframes and prototypes for the mobile and desktop

Results

The project concluded with the compilation of a report and guidelines for future development to the client. This project exemplifies my commitment to enhancing system usability, employing a user-centric approach, and delivering tangible design solutions for our clients.
